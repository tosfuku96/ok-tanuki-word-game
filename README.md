# ok-tanuki-word-game

「たぬきのOKことば工房」は、OK自立支援事業所 久喜の狸キャラクターを使った、やさしい文字ならべミニゲームです。

## 起動方法

1. このフォルダを開きます。
2. `index.html` をブラウザで開きます。
3. 「はじめる」ボタンを押して遊びます。

サーバーや外部ライブラリは不要です。ローカルブラウザだけで動きます。

## iPhoneでアプリ風に使う方法

同じWi-Fi内でPCから簡易サーバーを起動します。

```powershell
cd C:\CodexWork\ok-tanuki-word-game
python -m http.server 8000
```

PCのIPv4アドレスを確認します。

```powershell
ipconfig
```

iPhoneのSafariで `http://PCのIPv4アドレス:8000` を開き、共有ボタンから「ホーム画面に追加」を選びます。

本格的にPWAとしてオフライン対応まで使う場合は、HTTPS環境での公開が必要です。

## ファイル構成

```text
ok-tanuki-word-game/
├─ index.html
├─ style.css
├─ script.js
├─ manifest.json
├─ sw.js
├─ README.md
└─ assets/
   └─ ok_tanuki.png
```

狸画像は `assets/ok_tanuki.png` に配置します。画像が存在しない場合でも、画面には代替表示が出ます。

## ゲーム内容

- 全10問の文字ならべゲームです。
- バラバラになった文字をクリックして、正しい言葉を完成させます。
- 正解すると「OK！よくできました！」と表示されます。
- 不正解でも「だいじょうぶ、もう一回やってみよう！」と表示されます。
- 最後にスコアと結果メッセージを表示します。
